# Calculator API

## API - Examples

-   https://simple-calculator-api.herokuapp.com?equation=8/2(2%2B2)
-   https://simple-calculator-api.herokuapp.com?equation=8^-2

## Support Operators

-   Addition `+`
-   Subtraction `-`
-   Multiplication `*`
-   Division `/`
-   Exponentiation `^`
-   Brackets, `(` and `)`

## Other features

-   Each statement can contain more than 1 operator
-   Negative numbers
